# Data

The notebooks in `examples/` depend on three publicly available datasets, none of which are redistributed in this repository. Each notebook cites the dataset in its first markdown cell. Download instructions below.

## Datasets

### CheXmask VinDr-CXR segmentations

Used by `examples/cxr_segmentation_distortion.ipynb`.

- **Source:** Gaggion et al., *CheXmask: a large-scale dataset of anatomical segmentation masks for multi-center chest x-ray images*, Scientific Data, 2024 (arXiv:2307.03293).
- **Download:** https://physionet.org/content/chexmask-cxr-segmentation-data/
- **License:** PhysioNet Credentialed Health Data License 1.5.0; cite the original work.
- **Expected file:** `VinDr-CXR.csv` placed at the repository root. The notebook reads it and writes a derived feature cache to `data/cxr_features_cache.csv`.
- **Size:** ~900 MB raw; ~3 MB cache after first run.

### PTB-XL ECG

Used by `examples/ecg_subgroup_distortion.ipynb`.

- **Source:** Wagner et al., *PTB-XL, a large publicly available electrocardiography dataset*, Scientific Data 7, 154 (2020).
- **Download:** https://physionet.org/content/ptb-xl/
- **License:** Open Data Commons Attribution License v1.0.
- **Expected path:** `data/ptbxl/ptb-xl-a-large-publicly-available-electrocardiography-dataset-1.0.3/` (the notebook reads `ptbxl_database.csv` and the `records100/` WFDB signal files from that directory).
- **Size:** ~2 GB raw.

### US Census ACS PUMS

Used by all `*_calibration.ipynb` and `*_wealth*.ipynb` notebooks.

- **Source:** U.S. Census Bureau, American Community Survey 1-Year Public Use Microdata Sample.
- **Download:** https://www.census.gov/programs-surveys/acs/microdata/access.html
- **License:** Public domain (U.S. federal data).
- **Expected paths:**
  - `data/pums/psam_p10.csv` (Delaware person file)
  - `data/pums/psam_h10.csv` (Delaware household file)
  - `data/pums/psam_husa.csv` + `data/pums/psam_husb.csv` (national household files, two-part)
  - `data/pums/psam_pusa.csv` + `data/pums/psam_pusb.csv` (national person files, two-part)
- **Size:** ~3 GB raw across the national files.

## Reproducibility

The notebooks read from these locations. The cached feature files (e.g. `data/cxr_features_cache.csv`) are generated by the notebooks themselves on first run and gitignored thereafter.

Nothing in `data/` is tracked by git. The `.gitignore` excludes the entire `data/` tree and the root-level source CSVs.
